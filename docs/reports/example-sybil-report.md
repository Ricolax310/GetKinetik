## How this scan works

Four conservative heuristics. Each one would, on its own, be defensible in front of an auditor — none of them rely on guessing intent.

1. **Co-location clusters** — ≥5 nodes within 50m of each other.
2. **Birth bursts** — ≥10 nodes first observed inside the same 60-minute window.
3. **Beat-rate impossibility** — claimed lifetime heartbeats per second > 1.0.
4. **Metadata twins** — ≥3 nodes sharing an exact (firmware, hardwareId) fingerprint.

Honest single-operator clusters (e.g. a coffee shop running two devices) do not trigger these thresholds. The bureau errs on the side of false negatives over false positives.
