## USPTO Trademark Filing Guidance

**Status:** Not yet filed. File before any acquisition term sheet.

**Mark to file:** GETKINETIK
**Class(es):**
- Class 9 — Computer software for cryptocurrency and digital token management; mobile application software for decentralised physical infrastructure network aggregation
- Class 42 — Software as a service (SaaS); providing online non-downloadable software for blockchain and cryptocurrency services

**Filing type:** TEAS Plus (online, lower cost)
**Cost:** $250–$350 per class (USPTO filing fee, 2025 rates)
**Processing time:** 8–12 months to registration
**Attorney recommended:** Yes — a trademark attorney review before filing reduces rejection risk

**Key steps:**
1. Conduct a TEAS clearance search at `tmsearch.uspto.gov` — confirm "GETKINETIK" is not already registered in Class 9/42.
2. File via `teas.uspto.gov` using TEAS Plus.
3. Designate OutFromNothing LLC as the owner (not Eric Guthmann personally).
4. Use a bona fide use-in-commerce basis (you already have an app and website in use).

**Why file before M&A:**
- A registered trademark significantly increases acquisition price.
- Buyers in the identity/crypto space (Coinbase, Polygon, Worldcoin) expect clean IP.
- A pending application (even unregistered) creates a priority date that protects against copycats during the sale process.
